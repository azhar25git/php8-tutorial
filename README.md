# Modern PHP 8.x Masterclass

A single-page, modern guide to PHP 8.x programming — from core fundamentals to enterprise engineering and production deployment.

**Live site:** [azhar25git.github.io/php8-tutorial](https://azhar25git.github.io/php8-tutorial/)

## What's inside

- 32 topics across 9 modules
- Idiomatic PHP 8+ code examples
- Common anti-patterns and how to avoid them
- Security-first, production-ready guidance

## Tech stack

- HTML + Tailwind CSS
- Alpine.js for navigation and theming
- GitHub Pages for hosting

## Running locally

Open `index.html` in your browser, or serve it with any static server:

```bash
php -S localhost:8000
```
